# Tetris Palaclone

Un clon del clásico **Tetris** hecho en JavaScript puro para navegador.

## 🎮 Jugar ahora

👉 https://paladroncillo.github.io/tetris-palaclone/

> Si el link aún no abre, activa GitHub Pages en:
> **Settings → Pages → Deploy from a branch → main / root**

## Controles (teclado)

- **⬅️ / ➡️**: mover pieza
- **⬆️**: rotar
- **⬇️**: bajar suave
- **Espacio**: caída rápida
- **P**: pausa

## Controles táctiles

Incluye botones para jugar en móvil:

- ◀ izquierda
- ⟳ rotar
- ▶ derecha
- ▼ bajar
- ⤓ caída rápida
- ⏯ pausa

## Funcionalidades

- Sistema de puntaje, nivel y líneas
- Vista previa de siguiente pieza
- Efectos de sonido (con botón ON/OFF)
- Pantalla de **Game Over** con puntaje final y reinicio

## Ejecutar localmente

Solo abre `index.html` en tu navegador.

Opcional con servidor local:

```bash
python3 -m http.server 8080
```

Luego abre `http://localhost:8080`.
